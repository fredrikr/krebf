# krebf
A Z-code interpreter in Ruby, (c) Fredrik Ramsberg 2026, MIT License

**Aim**: Fully support v1-7 and v8 Z-code games. Hopefully, it won't have any dependencies, except for a standard Ruby installation. It shouldn't matter if you're on Windows, Linux or Mac.

**Status**: z1-z3 support is pretty much complete. Restart, MORE prompts, stream handling and split-screen aren't in place yet.
