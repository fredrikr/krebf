# krebf
A Z-code interpreter in Ruby, (c) Fredrik Ramsberg 2026, MIT License

**Aim**: Full support for v1-v5, v7 and v8 Z-code games. Hopefully, it won't have any dependencies, except for a standard Ruby installation. It shouldn't matter if you're on Windows, Linux or Mac.

**Status**: Support for v1-v3 Z-code games is pretty much complete. MORE prompts, statusline, stream handling and split-screen aren't in place yet. Accented character support is currently sketchy.
